Terminal Contact App

A simple terminal-based contact management application written in JavaScript (Node.js).

📦 Features

- ✅ Add new contacts
- 🔍 Search for contacts
- 📋 List all contacts
- ❌ Delete contacts

## 🛠 Requirements

- Node.js (v14 or above recommended)
